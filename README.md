# Duolingo Explorer

Duolingo Explorer is a front-end web project inspired by the Duolingo language learning platform.  
This project is designed to explore language learning features with a clean and user-friendly interface.

---

## Project Description

Duolingo Explorer is a responsive website that introduces users to language learning in a fun and interactive way.  
It showcases different languages, learning sections, and a modern UI similar to Duolingo.

This project is created for academic purposes and front-end practice using web technologies.

---

## Objectives

- To practice HTML, CSS, Bootstrap, and JavaScript
- To design a clean and user-friendly UI
- To understand responsive web design
- To create a real-world inspired web project

---

## User Roles

- Visitor
  - View available languages
  - Explore learning sections
  - Navigate the website

##  Features

- Responsive design using Bootstrap

1. Download or clone the repository
2. Open the project folder
3. Double-click index.html
